Core Features

Everything you need to manage transfers

A complete toolkit for agents and administrators to record, track, and report every transaction.

💸

Transaction Recording

Log Send and Withdrawal operations with full sender/receiver details, amount, fees, route, and payment method. Status updates in real time.

👥

Agent Management

Administrators create, activate, suspend, and monitor agents. Each agent operates under a unique code tied to a country and commission rate.

📊

Statistics & Charts

Interactive Chart.js dashboards — monthly volumes, country breakdowns, top routes, completion rates — for both admins and individual agents.

🌍

Multi-Country Support

8+ Sub-Saharan African countries with their flags, currencies, phone codes, and default fee percentages configured out of the box.

📤

CSV Export

Download complete transaction history as CSV at any time — full export for admins, personal export for each agent.

📨

Agent Reports

Agents can send messages and reports directly to administrators through the built-in messaging system, visible in the admin panel.

How It Works

Up and running in 4 steps

From installation to your first recorded transfer.

1

Install & Configure

Clone the repo, run composer install, set your .env with your MySQL credentials and run migrations with START_BLUESKY.bat.

2

Log In as Admin

Use the seeded administrator account to access the full admin panel — manage agents, countries, and settings from day one.

3

Create Agents

Add and activate agent accounts. Each agent gets a unique code, is assigned to a country, and has their own secure dashboard.

4

Record Transfers

Agents log every transaction with full sender, receiver, route, amount, and fee details. The admin dashboard updates in real time.
