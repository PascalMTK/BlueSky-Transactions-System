Core Features
💸

Transaction Recording

Log Send and Withdrawal operations with full sender, receiver, amount, fees, route and status details.

👥

Agent Management

Create and monitor agents. Each has a unique code, country assignment and their own secure dashboard.

📊

Statistics & Charts

Interactive Chart.js dashboards — monthly volumes, country breakdowns, and completion rates.

📤

CSV Export

Download full transaction history as CSV — complete export for admins, personal for each agent.
